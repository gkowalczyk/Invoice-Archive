#                             A simple application for archiving invoices


**Technology Backend:**

- Spring Boot
- REST API
- Lombok

**Technology Frontend:**

- Thymeleaf


###  Run project (IntelliJ IDEA)

After download repository:

```bash
click button maven compile
```
```bash
java -jar ./target/springmvc-0.0.1-SNAPSHOT.jar
```
Run browser
```bash
localhost:8080
```
